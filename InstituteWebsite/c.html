What is C? C is a general-purpose, procedural, high-level programming language used in the development of computer software and applications, system programming, games, and more. C language was developed by Dennis M. Ritchie at the Bell Telephone Laboratories in 1972. It is a powerful and flexible language which was first developed for the programming of the UNIX operating System. C is one of the most widely used programming languages.
